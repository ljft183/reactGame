# Game 
 
## 설명

시작 버튼을 눌러 나오는 1~16 까지의 수를 순서대로 얼마나  
빠른 속도로 누르는지 비교하는 게임입니다.  
  
게임 완료 시 점수 등록할 수 있으며 타인의 점수와 비교할 수  
있습니다.
